# AI Portfolio (React + Node.js)

A full-stack portfolio application featuring a React frontend with advanced 3D graphics and a Node.js/Express backend.

## Structure

- **client/**: Vite React application (Frontend)
- **server/**: Express.js application (Backend)

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm run install:all
    ```

2.  **Run Development:**
    ```bash
    npm start
    ```
    This will run both the client (port 5173) and server (port 5000) concurrently.

## Features

- **3D Neural Network Background**
- **Holographic Tilt Cards**
- **System Status Dashboard**
- **API Proxying**
